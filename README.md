# Kepler-Lab 🌌

¡Bienvenido/a a mi **viaje de limpieza y exploración de datos**!  

Este es mi repositorio del **Proyecto 2 – Kepler**, donde voy aprendiendo Python y cómo preparar datos para analizarlos sin morir en el intento 😅. 


---

## 🎯 Qué busco aquí
- Aprender a limpiar y organizar datasets.  
- Entender cómo funcionan las estructuras de datos en Python (listas, diccionarios, tuplas…).  
- Practicar condicionales, bucles y funciones paso a paso.  
- Tener todo documentado en mi **bitácora de aprendizaje**.  

Todo esto como preparación para proyectos más grandes y análisis de datos reales 🚀.
