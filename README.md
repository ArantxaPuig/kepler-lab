# 🌌 Kepler-Lab

¡Bienvenido/a a mi **viaje de limpieza y exploración de datos**!  

Este repositorio corresponde al **Proyecto 2 – Kepler**, donde estoy aprendiendo Python mientras preparo datasets para analizarlos… sin perder la cabeza 😅.

---

## 🎯 Objetivos

En este proyecto busco:  

- 🧹 Aprender a **limpiar y organizar datasets**.  
- 🐍 Entender las **estructuras de datos en Python**: listas, diccionarios, tuplas…  
- 🔄 Practicar **condicionales, bucles y funciones** paso a paso.  
- 📖 Mantener una **bitácora de aprendizaje**, para registrar errores, descubrimientos y aprendizajes.  

Todo esto es preparación para **proyectos más grandes y análisis de datos reales** 🚀.
